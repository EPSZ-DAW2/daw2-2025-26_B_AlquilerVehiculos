# Capa de Datos (Modelos) - Rent-A-Car
**Integrantes:** DHIAEDDINE JEBABLI y AHMED METTITI 

## Tareas Implementadas:
- **Modelo E-R:** Diseño e implementación de la base de datos relacional definiendo entidades clave: Vehículos, Clientes, Reservas, Categorías y Multas.
- **Baja Lógica:** Los vehículos vendidos o dados de baja no se eliminan físicamente para mantener la integridad en el historial de alquileres.
- **Cálculos SQL:** Algoritmo de cálculo de coste total automático basado en los días de alquiler y el precio de la categoría.
- **Procedimientos:** Implementación de lógica para validación de disponibilidad de flota y evitar solapamiento de fechas.
